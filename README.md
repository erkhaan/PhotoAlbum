# PhotoAlbum

## Details of the project
* Developed in Xcode 12.4, Swift 5.1, iOS 14.4
* MVC with Coordinator
* UI Kit (code-only)
* FBLoginSDK for login button
* Xcoordinator, Alamofire, SnapKit
